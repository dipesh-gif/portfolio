Tic Tac Toe Game (2023)

Created a responsive Tic Tac Toe game using HTML, CSS, and JavaScript. The game allows two players to alternate turns, with the first player starting with "X" and followed by "O". The app checks for win or draw conditions and shows a message when the game ends. Players can reset the game at any point. The background dynamically changes based on the outcome for a more immersive experience.

The game alternates turns between the players, with automatic win detection and result display. A reset button is available to start a new round after a win or draw.

To-Do List Application (2023)

Developed a To-Do List app using HTML, CSS, and JavaScript. Users can add tasks, display them in a list, and remove them as needed. The app features a datetime-local input for setting deadlines, which are displayed in a user-friendly format. The app is fully responsive, ensuring a smooth experience across different devices.

Users can manage tasks by adding, removing, and setting deadlines. The interface is designed to be clean, intuitive, and accessible across multiple devices.
